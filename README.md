# Requirements
* node.js >= v0.12.x
* npm >= v1.4.x

Please install this libraries.
```
npm install ntwitter
```

```
npm install socket.io
```

```
npm install express
```

# Usage

How to use this application?

```
node server.js
```

Access following a local loopback address.

localhost:5000 or 127.0.0.1:5000
